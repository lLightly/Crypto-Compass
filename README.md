# СИК — «Компас»

**Система интерпретации крипторынка** — интеллектуальный дашборд для анализа **BTC** и **ETH** на основе:

- Отчётов **COT** (Commitment of Traders) от CFTC
- Индекса **VIX** (режим риска)
- Макроиндикаторов (SPX, DXY, US10Y)
- Z-score и композитного COT-индекса

Проект автоматически генерирует **вердикты** («Бычий тренд» / «Медвежий тренд» / «Нейтральный»), считает **суммарный балл** и показывает историческую точность стратегии.

---

## ✨ Возможности

- **BTC Dashboard** и **ETH Dashboard** — цена, VIX, COT-индекс, Z-score, чистые позиции
- **Macro Context** — корреляция BTC/SPX и «вакуум ликвидности» (BTC + DXY + US10Y)
- **Conclusion** — объективный вердикт по рынку + объяснения
- **Trend Validation** — полное историческое тестирование стратегии с кривой капитала, CAGR, Sharpe, Calmar, Max DD и точностью тренда
